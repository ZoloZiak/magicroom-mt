## Čo chceš upraviť?

| Návod | Súbor |
|-------|-------|
| Svadobné šaty | `content/json/dresses.json` + fotky do `content/images/dresses/` |
| Služby a ceny | `content/json/services.json` |
| Partneri | `content/json/partners.json` |
| FAQ (často kladené otázky) | `content/json/faqs.json` |
| Dekorácie na prenájom | `content/json/decor.json` |
| Galéria (fotky) | `content/json/gallery.json` + fotky do `content/images/gallery/` |

## Všeobecný postup

1. Otvor súbor v GitHub
2. Klikni na ✏️ (Edit)
3. Uprav JSON
4. Dole klikni **Commit changes**
