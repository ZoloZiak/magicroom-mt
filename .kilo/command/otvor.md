Spusti dev server pomocou `npm run dev` v pozadí (bez blokovania).
Potom načítaj štruktúru projektu:
1. Prečítaj AGENTS.md pre inštrukcie projektu
2. Globni všetky src/**/*.ts, src/**/*.astro súbory pre pochopenie štruktúry
3. Prečítaj package.json pre skripty a závislosti

Výsledok: stručný súhrn projektu (názov, stack, dostupné skripty, počet stránok/komponentov).
