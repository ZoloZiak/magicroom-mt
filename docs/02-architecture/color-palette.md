## História farieb

**2026-04-01:** Pôvodné farby — hnedá škála (`#8B6F5C`, `#D4B8A0`, `#2D2421`)

**2026-04-02:** Zmena na ružovú podľa reálneho salónu:
- Natália: "namiesto tej hnedej by som tam supla asi takú tu ružovú"
- Photo analysis: dominantné ružové tóny (`#c78aa8`, `#ac607d`)
- primary: `#8B6F5C` → `#B87A8E` (ružová)
- secondary: `#D4B8A0` → `#E8D0D8` (svetlá ružová)
- primary-900: `#2D2421` → `#3D2A2E` → `#4A2E3A` (bordová)

**Kľúčové pravidlo:** Nikdy nepoužívať staré hnedé tiene `rgba(45,36,33,...)` — len ružové `rgba(74,46,58,...)` alebo `rgba(53,31,42,...)`
