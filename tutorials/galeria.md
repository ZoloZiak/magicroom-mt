# Galéria — ako pridať fotky

🔗 **Otvoriť:** `data/gallery.json`

---

## Pridať fotku do zoznamu

```json
{
  "gallery": [
    {
      "filename": "moje-fotka.jpg",
      "alt": "Popis obrázka",
      "altEn": "Image description",
      "title": "Názov",
      "titleEn": "Title"
    }
  ]
}
```

---

## Nahrať skutočnú fotku

1. Choď do `public/images/`
2. Nahraj súbor `moje-fotka.jpg`

---

## Poznámky

- `filename` = názov súboru v `public/images/`
- `alt` = popis pre SEO (slovensky)
- `altEn` = popis pre SEO (anglicky)
