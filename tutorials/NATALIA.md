# Návody pre Natáliu

🔗 **GitHub:** https://github.com/ziak-z/magicroom-mt

---

## Čo chceš upraviť?

| Návod | Súbor |
|-------|-------|
| Svadobné šaty | `data/dresses.json` + fotky do `public/content/dresses/` |
| Služby a ceny | `data/services.json` |
| Partneri | `data/partners.json` |
| FAQ (často kladené otázky) | `data/faqs.json` |
| Dekorácie na prenájom | `data/decor.json` |
| Galéria (fotky) | `data/gallery.json` + fotky do `public/images/` |

---

## Všeobecný postup

1. Otvor súbor v GitHub
2. Klikni na ✏️ (Edit)
3. Uprav JSON
4. Dole klikni **Commit changes**
